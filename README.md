# Portfólio de Tatuagem

Site de portfólio profissional para tatuador, com foco em apresentação visual e contato rápido via WhatsApp.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Layout responsivo
- Menu mobile
- Galeria de trabalhos com lazy load
- Lightbox para visualização de imagens
- Formulário que envia mensagem direta para o WhatsApp
- Botão flutuante de WhatsApp

## Como usar
Clone o repositório e abra o `index.html` no navegador.
